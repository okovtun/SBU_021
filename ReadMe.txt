Стишок должен быть выровнен по середине, куплеты должны быть разделены пустой строкой.

----------------------------------------

2. С клавиатуры вводится два числа - основание степени и показатель степени;
3. Вывести на экран таблицу ASCII-символов, разделенных пробелом, по 16 символов в строке;
4. С клавиатуры вводится число. Программа должна сказать является ли это число палиндромом, или не является 12321
